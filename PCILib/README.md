## PCILib内有什么?

1. [PCILib](./__init__.py) 总库
2. [PCIhashLib](./PCIhashLib.py) hash相关
3. [PCImathLib](./PCImathLib/__init__.py) 数学计算
4. [PCItextLib](./PCItextLib.py) 文本处理

## 已经实现的功能

* hash值计算
* 繁简中文转换
* Fourier级数、变换,Laplace变换
* 删除重复文件
* 删除空目录
* 批量重命名
* 批量复制、移动
* 批量文件校验
